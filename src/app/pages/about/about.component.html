About

This sample app lets users create orders for dishes from a provided menu. Links to source codes can be found in the footer.

To manage Orders, Categories and Dishes:

  • Sign in as 'Admin' by clicking on login button in the top right corner.
  • Navigate to 'Dashboard' from the top nav.

Technologies

Front-End

  • The app is built as a single-page application using Angular 12 and Angular Material 12.
  • Global state of the app is managed by NgRx 12.
  • Unit testing is done via Karma and Jasmine
